  • loadAsync(themeUrl: string): Promise<Theme>
    • Defined in harp-mapview/lib/ThemeLoader.ts:24

    Loads a Theme from a remote resource, provided as a URL that points to a JSON-encoded theme.

    Relative URLs are resolved to full URL using the document's base URL (see: https://www.w3.org/TR/WD-html40-970917/htmlweb.html#h-5.1.2).

      The URL to the theme.
