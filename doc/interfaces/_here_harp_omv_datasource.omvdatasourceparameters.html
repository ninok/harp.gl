Optional filterDescr

filterDescr: OmvFeatureFilterDescription
  • Defined in harp-omv-datasource/lib/OmvDataSource.ts:119

A description for the feature filter that can be safely passed down to the web workers. It has to be generated with the help of the OmvFeatureFilterDescriptionBuilder (to guarantee correctness). This parameter gets applied to the decoder used in the OmvDataSource which might be shared between various OmvDataSources.
